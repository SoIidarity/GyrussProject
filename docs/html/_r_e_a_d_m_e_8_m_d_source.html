9 # Game Features and Development
10 ## *Whats New*
11 * Game plays fullscreen, 1920x1080
12 * Added Perlin noise implementations
13 * Start splash screen tribute to the original game
14 * Added additional animation to splash screen
15 * Added an additional enemy variant: Wanderer, using 2D perlin noise
16 * Adjusted Game speed to start slower
17 * Increased max enemies at any time
18 * Added Victory condition score to HUD: enemy deaths count down from 100
19 * Engine refactoring for seperation of concerns
20 * Technical documentation now included with game ( in docs/html.index.html )
21 
22 ### SplashScreen:
23 * Player ship image now floats about (procedurally animated with Perlin noise)
24 
25 ### PlayerShip/HUD:
26 * PlayerShip state in HUD now reflects weapon upgrade
27 
28 ### Entity behavior:
29 * Entities no longer group horizontally.
30 * Sattelite texture changes to be more visually distinct
31 * Additional enemy variant (Wanderer) with new texture
32 
33 ### Object-Oriented design:
34 * All audio, including music, now the responsibility of Soundcontroller
35 * Removed player move sound from PLayerShip (No classes derived from Entity have any sound methods at all)
36 * Entity now inherits from two Base classes: Moveable and Animatable
37 * Moveable Class: All movement and translation/dimensional members and methods
38 * Animatable Class: All texture and tile-set animation members and methods
39 
40 ### Technical documentation
41 * Nicer enum formatting (including entity texture images in documentation)
42 * All objects documented (complete)
43 
44 ### Other Application features:
45 * New Highscore!
46 * Improved Game art assets and sound.
47 * Developer mode (in OSX only) has permenant ivulnerability mode
48 * Added "Spawn Enemy" developer keys
49 * Developer keys (in debug mode only) : <br> [ : slow down game speed <br>
50  ] : speed up game speed <br>
51  l : upgrade gun <br>
52  k : kill (lose player life) <br>
53  p : godmode (invulnerability)
54  o : turn off invulnerability
55
