514 // A 24 byte string class (can be as small as 17 for x64 and 13 for x86) that can hold strings with length
515 // of up to 23 chars on the stack before going on the heap - the last byte of the buffer is used for:
516 // - "is small" bit - the highest bit - if "0" then it is small - otherwise its "1" (128)
517 // - if small - capacity left before going on the heap - using the lowest 5 bits
518 // - if small - 2 bits are left unused - the second and third highest ones
519 // - if small - acts as a null terminator if strlen() is 23 (24 including the null terminator)
520 // and the "is small" bit remains "0" ("as well as the capacity left") so its OK
521 // Idea taken from this lecture about the string implementation of facebook/folly - fbstring
522 // https://www.youtube.com/watch?v=kPR8h4-qZdk
523 // TODO:
524 // - optimizations - like not deleting memory unnecessarily in operator= and etc.
525 // - resize/reserve/clear
526 // - substr
527 // - replace
528 // - back/front
529 // - iterator stuff
530 // - find & friends
531 // - push_back/pop_back
532 // - assign/insert/erase
