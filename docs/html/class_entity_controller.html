Detailed Description

EntityController Class.

Responsible for spawning and destroying non-player game entities (bullets, enemies, meteoroids, explosions) Responsible for the behaviour and logic of all non-player entities Responsible for collision detection between these entities and playerShip Responsible for updating the movement of all non-player entities

Constructor for the EntityController class. Called directly from game object to handle the creation, destruction and drawing of all game objects.

Controller and manager for game entity objects.

On every game frame, respective functions are called to evaluate game entity states, such as spawning new enemies, collisions and enemy bullet shooting

Parameters
resolutionThe Screen resolution of the game
playerShipThe playerShip, generated from the game.cpp. needed for collision detection
texturesused to create new entities that require this class for creation
scoreobject to track the current game score, enemies killed and player deaths. created in game.cpp
speedModifierdefines how fast the game runs at a particular time to speed up game as time progresses
