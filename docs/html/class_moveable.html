Pure Virtual method to set the future movement, either by change in angle or change in distance.

Some entities only need a change in angle, or distance, not both. The actual position of the entity is not altered until the move() method is called, which performs the actual translation

inlinevirtual

Virtual method to set up the next frame's position for the object, with a different screen origin.

The actual position of the entity is not altered until the move() method is called, which performs the actual translation

Parameters
angleThe angle the entity should move for the next frame (0-360, with 0 being the positive horizontal x axis, with the specified centre argument as the origin)
distanceHow far the entity should move for the next frame (in screen space)
centreThe coordinate {x,y} to be considered as the origin for this move
