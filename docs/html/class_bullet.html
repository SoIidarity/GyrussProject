Constructor for the Bullet Class. Calls the ABC constructors for the inheritance hierarchy, Entity, Moveable and Animatable.

Projectile bullet Class.

Bullets are spawned by shoot events, and then move at a fixed speed along radial lines towards or away from the centre of the screen.

Parameters
resolutionThe Screen resolution of the game
distanceFromCentreThe radius from centre at which the bullet should be created
angleThe argument (angle) at which the bullet should be placed upon creation
scaleThe initial size of the bullet object (also influenced by the sprite texture, and scaled by distance from the origin in the move() command.
typeAn enum that identifies the Entity type. Defined in common.hpp, passed to the Animatable base class
textureHolderThe textureHolder object, passed by const ref. provides a unique_ptr to an sf::Texture for the sprite data member
idThe unique ID identifying the texture for the Entity
